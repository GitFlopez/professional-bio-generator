### Quality Notes

**AVOID:**
- "Passionate about cybersecurity" (everyone says this)
- "Thought leader in the security space" (vague and self-bestowed)
- "Results-driven security professional" (meaning-free)
- "Leveraging cutting-edge solutions" (corporate filler)
- "Proven track record" (implied by the numbers — say the numbers)

**POWER WORDS:** Protect → Harden / Lead → Drive / Implement → Build / Experienced → Specialized / Innovative → Specific methodology or framework name / Skilled → Certified + years / Team player → Cross-functional / Strategic → [specific strategy name] / Passionate → [specific outcome instead] / Utilize → Use

**HOOK RATINGS:**
- 50-word 1st person opener: "I help mid-market companies survive the security incident they haven't had yet." — 9/10. Provocative, specific. Would improve if it named the industry: "...survive the security incident their industry hasn't prepared them for."
- 150-word opener: "Most security breaches start with a patient attacker and an unprepared organization." — 8/10. Creates immediate tension. Could sharpen: "Most breaches don't start with sophisticated attackers. They start with patient ones and unprepared teams."
- 300-word opener: "Most security breaches aren't sophisticated. They're patient." — 9/10. Short sentences. Perfect pacing. Keep as-is.
- 1st person opener: "The call came at 2 AM on a Tuesday." — 10/10. Scene-setting, specific, immediate tension.
